# NIYA-SHOP

A responsive e-commerce website built with React, TypeScript, and Vite.
The project allows users to browse products, manage a shopping cart using React Context API, and persist cart data with Local Storage.

## Features

- Browse products
- Add and remove products from the shopping cart
- Manage cart state using React Context API
- Persist cart data with Local Storage
- User login functionality
- Responsive user interface
- Reusable React components

## Tech Stack

- React
- TypeScript
- Vite
- React Context API
- Local Storage
- CSS

## Installation
```bash
npm install
```
## Run the Project

Start the development server:
```bash
npm run dev
```
In a separate terminal, start the JSON server:
```bash
npx json-server data/db.json -p 8001
```
